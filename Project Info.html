Need for Speed


List of electronic devices that we used for the project:

Section 1: Hardware Components

  • Arduino

    • We used Arduino mega 2560 to write and upload computer code to the car.

  • 12V Battery

    • For the car project, we used 12 V chargeable battery.


  • Bluetooth

    • For the RC car, we used HC 06 bluetooth.

  • UR sensor

    • We used UR sensor for the obstacle recognition.

  • USB cable

    • We used USB cable to upload the code from laptop to our arduino.

  • Driver Motor Controller

    • We used L293 D to control the working speed and direction of 4 motors of the car.


    Section 2: Programming

    For our project, we used C++ language for the RC car and we used HTML and CSS for the NFS website.
